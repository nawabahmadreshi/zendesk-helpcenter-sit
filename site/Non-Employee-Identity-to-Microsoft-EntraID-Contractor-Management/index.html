The Aquera platform allows your organization to scale complex data management by automating data integration across resource types. Aquera's integrations facilitate the automatic provisioning, ensuring consistency across systems. It allows organizations to streamline processes, dynamically scale resources, and automate repetitive tasks, ultimately improving efficiency in managing intricate IT infrastructures and applications.

This integration synchronizes contractor records from the ACM (Aquera Contractor Management) system into Microsoft Entra ID, ensuring that contractor accounts in Entra always reflect the current state of contractor data in ACM. 

It reads contractors in batches, checks whether a matching Entra user already exists using the contractor’s email, and then either creates a new Entra user or updates the existing user using a defined contractor-to-Microsoft attribute map. A report-only mode allows the integration to simulate the actions without making changes, while still producing a detailed summary of users that would be created or updated.

In addition to handling active contractors, the integration includes a full offboarding process that identifies contractors who have been marked as deactivated in ACM. For these contractors, the integration locates the matching Entra user and disables their Entra account (accountEnabled = false) to prevent further access. This deactivation process supports both live execution and report-only mode, logs each action, and produces a CSV-based audit trail that records all processed contractors, skipped records, errors, and successful account disablements.

Throughout the entire workflow, the integration provides operational transparency and governance controls. Logging can be enabled to generate detailed execution reports, and optional email notifications can alert administrators about successes, failures, or critical errors. At the end of each run, the integration summarizes all activities—including created, updated, skipped, and disabled users—ensuring administrators have a complete view of provisioning and offboarding activity between ACM and Microsoft Entra.
