Overview

The Aquera platform allows your organization to scale complex data management by automating data integration across resource types. Aquera's integrations facilitate the automatic provisioning, ensuring consistency across systems. It allows organizations to streamline processes, dynamically scale resources, and automate repetitive tasks, ultimately improving efficiency in managing intricate IT infrastructures and applications.

This guide explains the end-to-end process of configuring the integration to assign and store a Universal Unique Identifier  for users in Dayforce and Active Directory by selecting preferred attributes during setup.

Prerequisites

The following are the prerequisites:

  • Admin Access to Active Directory: You must have admin access to your Active Directory instance. You can visit the Active Directory portal to get access to an Active Directory instance.
  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/.
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure the following application connectors in the Aquera Admin portal to integrate with Active Directory:
    • Dayforce: Ensure that you configure the Dayforce connector in the Aquera Admin portal. Refer to the Dayforce Configuration Guide for configuration details.
    • Active Directory: Ensure that you configure the Active Directory connector in the Aquera Admin portal.

Integration Setup

This section details the configuration and mapping process for integrating Dayforce with Active Directory.

Create Integration

Perform the following steps to create the integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Select Active Directory / Dayforce.
  5. Enter a name and a description for the integration.
  6. Select your Dayforce and Active Directory connector instances.
  7. Click Continue. The Configuration page is displayed to configure the integration.

Configuration

On the Configuration page, the following sections are displayed:

Applications

The Applications section displays the source and target application connectors for the integration. This section allows you to search and update these application connectors.

Parameters

The Parameters section enables you to customize the integration features and general settings according to your specific use case.

  • Enable Aquera Universal Unique Identifier: Select this checkbox to enable assigning a Universal Unique Identifier during the integration for selective attributes.

  • Select Dayforce Attribute for UUID: Select an attribute in Dayforce to store the Aquera Universal Unique Identifier.

  • Select Active Directory Attribute for UUID: Select an attribute in Active Directory to store the Aquera Universal Unique Identifier.

Global Settings:

  • Enable Report Generation: Select to generate the integration report.

Schedule

Integrations can be scheduled to run automatically based on your choice of timeframe. Refer to Integration Schedule Setup Guide for further details.
