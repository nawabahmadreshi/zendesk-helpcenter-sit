Overview

The Aquera platform enables your organization to scale complex data management by automating data integration across various resource types. Aquera's integrations facilitate the automatic provisioning, ensuring consistency across systems. It enables organizations to streamline processes, dynamically scale resources, and automate repetitive tasks, ultimately enhancing efficiency in managing complex IT infrastructures and applications.

This guide covers the end-to-end tasks in setting up the integration to sync Candidates  data from ADP Workforce Now to the ticketing resource in Zendesk.

Key Functions

The ADP Workforce Now to Zendesk integration automates the onboarding tickets with the following primary capabilities:

  • Creates a Zendesk onboarding ticket for each newly created candidate in ADP Workforce Now.

Prerequisites

The following are the prerequisites:

  • Admin Access to Zendesk: You must have admin access to your Zendesk instance. You can visit the Zendesk portal to get access to an Zendesk instance.
  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/.
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure the following application connectors in the Aquera Admin portal to integrate with Zendesk:
    • ADP Workforce Now: Ensure that you configure the ADP Workforce Now connector in the Aquera Admin portal. Refer to the ADP Workforce Now Configuration Guide for configuration details.
    • Zendesk: Ensure that you configure the Zendesk connector in the Aquera Admin portal.

Integration Setup

This section details the configuration and mapping process for integrating ADP Workforce Now with Zendesk. It outlines the steps to create the integration, set up integration preferences, and define how worker data is mapped from ADP to Zendesk.

Create Integration

Perform the following steps to create the integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Select ADP Workforce Now / Zendesk.
  5. Enter a name and a description for the integration.
  6. Select your ADP Workforce Now and Zendesk connector instances.
  7. Click Continue. The Configuration page is displayed to configure the integration.

Configuration

On the Configuration page, the following sections are displayed:

Applications

The Applications section displays the source and target application connectors for the integration. This section allows you to search and update these application connectors.

Parameters

The Parameters section enables you to customize the integration features and general settings according to your specific use case.

Integration Features

Enable the following features based on your requirements:

  • Enable Candidate Management: Select this checkbox to sync candidate records from ADP into the Zendesk onboarding ticket. When enabled, the Candidate Management section is displayed to configure integration settings for candidate management.

General Settings

Configure the following settings:

  • Enable Report-Only Mode: Select this checkbox to execute the integration in the report-only mode. The report-only mode generates a report to show the data that would be changed without actually making those changes.
  • Enable Logging: Select this checkbox to generate logs outlining data changes. When enabled, the Log Settings section is displayed for further configuration.
  • Enable Report Generation: Select this checkbox to generate the integration report. When enabled, the Report Settings section is displayed for further configuration.
  • Enable Email Notifications: Select this checkbox to enable integration-related notifications. When enabled, the Email Notification Settings section is displayed for further configuration.
  • Enable Log Insights: Select this to visualize the integration-related metrics. When enabled, the Log Insights Settings section is displayed for further configuration.

Candidate Management

The Candidate Management section enables you to customize the integration settings for candidate management according to your specific use case.
Note: This section appears only if you have selected the Enable Candidate Management setting in the Parameters section.

It has the following settings:

  • Select ADP Workforce Now Application Instance: Select the instance in which the candidates data is available and that triggers Zendesk to create tickets and workflows.
  • Enable Onboarding Ticket Creation: Enabling onboarding ticket creation will create a ticket in Zendesk for every matching candidate from ADP Workforce Now.
  • Candidate Filter Types: Select the required filter to fetch the candidates based upon the use case.
