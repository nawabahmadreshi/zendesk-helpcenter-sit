Overview

The Aquera platform allows your organization to scale complex data management by automating data integration across resource types. Aquera's integrations facilitate the automatic provisioning, ensuring consistency across systems. It allows organizations to streamline processes, dynamically scale resources, and automate repetitive tasks, ultimately improving efficiency in managing intricate IT infrastructures and applications.

This guide outlines the process for setting up an integration to generate a CSV report containing payroll journal entries and employee data from your ADP Workforce Now to Microsoft Dynamics 365 Business Central. The CSV report is then shared with the specified recipients via email.

Prerequisites

The following are the prerequisites:

  • Aquera Tenant Account: You must have a tenant account in the Aquera Admin portal. Visit https://admin.aquera.io to register for a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure the following application connectors in the Aquera Admin portal:
    • ADP Workforce Now: Refer to the ADP Workforce Now Configuration Guide for details.
    • Microsoft Dynamics 365: Refer to the Microsoft Dynamics 365 Configuration Guide for details.

Integration Setup

This section outlines the configuration required to send the generated CSV file to the designated users via email and to display the integration results on the dashboard. It covers creating the integration, configuring system preferences, scheduling the integration to run, and distributing the generated CSV to the designated users. 

Create Integration

Perform the following steps to create the integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Select ADP Workforce Now Workers to Microsoft Dynamics 365.
  5. Enter a name and a description for the integration.
  6. Select your ADP Workforce Now and Entra ID connector instances.
  7. Click Continue. The Configuration page is displayed to configure the integration.

Configuration

The Parameters section under Configuration enables you to customize the integration settings according to your specific use case. It has the following configuration parameters:

  • Report Only: Select this checkbox to generate a report that shows the data that would be changed, without actually making those changes. This setting can be selected for the first integration instance to identify any issues. You can clear this selection after verifying the report and confirming the expected result.
  • Generate Integration Report: Select the checkbox to generate the integration report.
  • Enable Dashboard Generation: Select to analyze the integration results in a dashboard.
  • Debug Mode: Select to generate logs outlining data changes that would occur for debugging.
  • Target Time Zone: 
  • Send Email Notifications: Select to enable sending the email notifications.
  • Send Integration Report via Email: Select to receive email notification with a CSV report for all integration runs.
  • Recipients' Email Address: Enter the CSV report's recipient email address. It accepts single or comma-separated addresses and must follow valid email formatting to ensure delivery. Incorrect values may result in a failed distribution of the report.
  • Email Subject on Successful Integration Execution: Enter the content that must be used as the email subject in the notifications sent for successful integration execution.
  • Email Subject on Failed Integration Execution: Enter the content that must be used as the email subject in the notifications sent for an unsuccessful integration execution.
  • Manager Filter Condition: Enter the manager's ID to filter employees.
  • Pay Cycle Frequency: Select the pay cycle frequency from the dropdown menu.
  • Pay Cycle Day: Select the pay cycle day from the dropdown menu.

Attribute Mapping

The ADP Workforce Now Workers to Microsoft Dynamics 365 template maps the attributes to generate a CSV report containing payroll journal entries.

There are four use cases:

  1. Employee punch-in and punch-out data is found. 
  2. Employee punch-in data is available, but punch-out data is not. Then the last punch-out data for the day is considered.
  3. Employee punch-in data is not found, but punch-out data is found. Such instances are ignored.
  4. Multiple punch-in and punch-out data are found. Then the first punch-in and last punch-out data for the day are considered.

Download CSV Report

You can download a report of an integration run in CSV format by clicking the execution status indicators in the History column and navigating to the Execution Timeline panel.

Run Integration

You can set up an event or schedule triggers to execute the integration. For further details, refer to the Integration Using Scripts guide.

Once the integration is successfully executed, Aquera syncs the workers created in ADP Workforce Now as users in Entra ID.
