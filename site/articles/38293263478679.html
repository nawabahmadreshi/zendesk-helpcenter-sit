Overview

This guide provides a step-by-step process for understanding and configuring the ADP Decidium application for all API Central Direct customers. Once the application is configured, you will be able to import, update, and search users in ADP Decidium

ADP API Central for ADP Decidium provides the following benefits:

  • Instantly access ADP APIs
  • Request and manage security certificates
  • Credential management
  • OpenID Connect and OAuth 2.0 authentication.

Prerequisite

You must have sufficient user permissions to access the ADP Decidium. 

To create new employees using the template configured for your project, the following steps are a prerequisite:

  1. Log on to ADP Decidium using admin credentials.
  2. Navigate to the Setup > Template Management > Hire/Rehire.
  3. Click the template that you want to use for employee creation.
  4. Click Assign Users.
  5. Ensure that the following checkboxes are selected:
    • API Central automated profile 
    • Don't inactivate or delete this profile. Membership changes could impact any existing integrations.

Connector Configuration

Perform the following steps to configure the ADP Decidium application:

  1. Navigate to Applications and click Add Application on the Aquera Admin Home page.

  2. Search for ADP Decidium - API Central, and click on the respective icon.

    Note: If you select ADP Decidium - API Central Direct, you must configure

    API Central Direct Attributes (Applicable only for ADP Decidium - API Central Direct) before you configure other tenant attributes as per your use case. 

  3. Enter the values for the following fields under the Main tab:
     
    • Display Name: Enter the display name for the application.
    • Description: Enter a description for the application.
    • Source Connection Type: Select Direct Connection.
    • Aquera Tenant Attributes: Refer to the Aquera Tenant Attributes section.
    • Source Owner: Enter a valid email ID of the source owner(s).
    • Subscribe error notification:  Select this checkbox to send notification emails to Source Owner(s) for any errors in user data transactions
    • Pick your Protocol: Select SCIM; this will populate the fields in the schema accordingly.
    • Copy this URL: Displays a URL. Copy this URL and paste it at the time of provisioning in the Identity Service Provider.
    • Authentication: The bearer token will be provided by the Aquera support team.
    • Schema: Consists of SCIM attributes and custom attributes for users.
  4. Click Create to add the ADP Decidium application.

Aquera Tenant Attributes

The following are the tenant attributes:

API Central Direct Attributes (Applicable only for ADP Decidium - API Central Direct)

The following tenant attributes are specific only to API Central Direct customers:

  • Client ID: Enter the client ID.
  • Client Secret: Enter the client secret.
  • Client Certificate: Upload the client certificate file.
  • Client Private Key: Upload the client private key file.
    Note: You need to retrieve the required values and files for the above tenant attributes by logging in to the API Central portal and performing the following steps:
  1.  
    1. Click the hamburger menu located at the top-left corner and click Projects.
    2. Click Create a project. The Create new project pop-up window is displayed.
    3. Enter the project name in the Project name text box. Ensure that 'Aquera' is mentioned in the name.
    4. Enter the project description in the Project description textbox.
    5. Click Next, then select a use case based on the data you are trying to access.
    6. Click Create Project. The project will be created, and the Projects page will be displayed.
    7. Click View project on your project to display the Credentials tab.
    8. Click Copy client ID under the Step 2 section to copy the client ID.
    9. Click the Show secret toggle button to activate the Copy client secret button.
    10. Click Copy client secret to copy the client secret.
    11. Click Manage Certificate under the Step 1 section and the Certificates page is displayed.
    12. Click Request certificate and Request new security certificate pop-up window is displayed.
    13. Click Next and provide the required details.
    14. Click Copy and paste it in a new file in your local system. Name the file as private.key and upload it in the Client Private Key tenant attribute. Click Next.
    15. Click Next to view the Generate Certificate page. Click Done to close the pop-up window and view the Certificates page.
    16. Click Download to download the .PEM certificate file and upload it in the Client Certificate tenant attribute.

Authentication

A bearer token is used for authentication. The Aquera Support team will provide the token.

How to Enable the Scopes for Your API Central Project

The following are the two methods to get the scopes enabled for your project:

  • Add the scopes to your API central project

    Login into the ADP API central portal.
    Click the ellipsis located at the top left corner and click Projects. The Projects page is displayed.

    Click View project on your project card. The selected project page is displayed.
    Click the APIs tab and click Add more APIs. The Discover APIs to add to your project page is displayed.
    Enter the API in the search box. The API is displayed.
    Click + Add to add it to your project.
