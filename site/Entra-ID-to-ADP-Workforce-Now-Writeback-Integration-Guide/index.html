Overview

The Aquera platform allows your organization to scale complex data management by automating data integration across resource types. Aquera's integrations facilitate the automatic provisioning, ensuring consistency across systems. It allows organizations to streamline processes, dynamically scale resources, and automate repetitive tasks, ultimately improving efficiency in managing intricate IT infrastructures and applications.

This integration enables you to synchronize contractors from NEIM to Active Directory and Entra ID. 

Key Functions

The NEIM / Hybrid integration performs the following functions:

  • Retrieves the data of contractors from NEIM and provisions them as users in Active Directory, which are then synced to Entra ID, based on the configured integration settings.
    • For new users, the integration generates a unique sAMAccountName, Common Name (CN), User Principal Name (UPN), and email address, depending on the configuration.
    • Generates a random password or a password based on a configured password policy for each new users.
    • Assigns users to Entra ID groups and Active Directory groups and Organizational Units (OUs) based on the assignment rules configured in the integration.
    • Assigns Entra ID licenses to users based on the assignment rules configured in the integration.
  • Updates or deactivates users in Active Directory and Entra ID based on the conditions configured in the integration.
  • Removes users from Active Directory and Entra ID groups based on the configured rules.
  • Removes license from Entra ID users based on the configured rules.
  • Generates reports and logs, and sends notification emails based on the integration settings.

Prerequisites

The following are the prerequisites:

  • Aquera Tenant Account: You must have a tenant account in the Aquera Admin portal. Visit https://admin.aquera.io to register for a tenant account. Contact your Aquera account manager to get Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure the following application connectors in the Aquera Admin Portal:
    • Aquera Contractor: For configuration details, refer to the Aquera Contractor Configuration Guide.
    • Active Directory:  For configuration details, refer to the Active Directory (AD) Version 8 Configuration Guide.
    • Entra ID: For configuration details, refer to the Microsoft 365 Configuration Guide or the Microsoft 365 App Registration Configuration Guide, depending on the type of authentication to be used for the connector.

Integration Setup

This section outlines how the integration must be configured to synchronize contractors from NEIM to Active Directory and Entra ID.

Create Integration

Perform the following steps to create the integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Select NEIM / Hybrid.
  5. Enter a name and a description for the integration.
  6. Click Continue. The Overview page is displayed to start configuring the integration.
