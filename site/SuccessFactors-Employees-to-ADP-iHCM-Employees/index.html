Overview

The Aquera platform allows your organization to scale complex data management by automating the integration of data across resource types. This guide covers the end-to-end tasks in setting up a data integration to sync the Employees resource of SuccessFactors to the Employees resource of ADP iHCM.

Prerequisites

The following are the prerequisites:

  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure following application connectors in the Aquera Admin portal to integrate with Microsoft Entra ID:
    • SuccessFactors: Refer to the SuccessFactors Configuration Guide for details.
    • ADP iHCM: Refer to the ADP iHCM Configuration Guide for details.

Data Integration Setup

Aquera's data integration automatically imports employees from SuccessFactors and provisions these employees in ADP iHCM.  It also syncs the payroll data of employees.

Aquera provides templates that are pre-configured integrations created for specific use cases. You can use these templates to create a data integration.

Perform the following steps to create a data integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to integrations on the left panel.
  3. Click Create integration.
  4. Select a pre-built data integration template based on your requirements. For this use case, select the template SuccessFactors Employees to ADP iHCM Employees.
  5. Enter a name and a description for the integration.
  6. Select the SuccessFactors connector in the Source Application dropdown field.
  7. Select the ADP iHCM connector in the Target Application dropdown field.
  8. Click Continue. A new page with the fields and tabs to configure the data integration is displayed.
  9. You can update the configurations if required. For more details, refer to the Data Integration Setup Guide. 
  10. Click Create to create the data integration.

Attribute Mapping

The Attribute Map tab allows you to map employees from SuccessFactors to the Payroll resource of ADP iHCM.

The following is a sample attribute mapping from SuccessFactors to ADP iHCM:

SuccessFactors Attribute / Value / Expression ADP iHCM Attribute
SOURCE.email businessCommunication.emails
SOURCE.hr.salary payDataInput.earnings
SOURCE.empCompensation.endDate payDataInput.payPeriodEndDate
SOURCE.empCompensation.startDate payDataInput.payPeriodStartDate
SOURCE.dateOfBirth person.birthDate
[{"formattedNumber" : SOURCE.perPerson.phoneNumbers[0].phoneNumber}] person.communication.mobiles
SOURCE.gender person.genderCode.codeValue
SOURCE.lastName person.legalName.familyName1
SOURCE.firstName person.legalName.givenName
SOURCE.salutation person.legalName.preferredSalutations
[{"expectedStartDate" : SOURCE.hireDate,"hireDate" : SOURCE.hireDate}] workAssignments
SOURCE.personKeyNav.personIdExternal workerID.idValue

In the source attribute field, you can include source attributes or write JavaScript expressions to map to the corresponding target attribute. The target attributes marked with a red asterisk (*) are mandatory to be mapped.

Running the Data Integration

You can click the Execute button at the top-right corner to manually run the data integration. You can also schedule the data integration to run automatically at specified intervals by configuring a frequency in the Schedule tab.

You can navigate to Logs  > Integrations in the Aquera Admin portal to view the integration execution logs.

After the data integration is successfully executed, new employees will be created and existing employees will be updated in the ADP iHCM application.
