Overview

The Aquera platform allows your organization to scale complex data management by automating the synchronization of data across resource types. This guide covers the end-to-end tasks in setting up a script synchronization to sync the Workers resource of ADP Workforce Now to the Users resource of Google Workspace.

Prerequisites

The following are the prerequisites:

  • Admin Access to Google Workspace: You must have admin access to your Google Workspace instance. You can visit the Entra ID portal to get access to an Entra ID instance.
  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure following application connectors in the Aquera Admin portal to integrate with Google Workspace:
    • ADP Workforce Now: Refer to the ADP Workforce Now Configuration Guide for details.
    • Google Workspace: Refer to the Google Workspace Configuration Guide for details.

Synchronization Setup

This section details how the synchronization must be configured to allow data push from ADP Workforce Now to Google Workspace.

Create Script Synchronization

Perform the following steps to create a new script synchronization:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Synchronizations from the left side panel.
  3. Click Create Synchronization.
  4. Select ADP Workforce Now Workers to Google Workspace Users.
  5. Enter a name and a description for the synchronization.
  6. Select your ADP Workforce Now and Google Workspace connector instances.
  7. Click Continue. A new page is displayed to configure the synchronization.

Run Script Synchronization

You can set event or schedule triggers to run the script synchronization. For more details, refer to the Executing a Synchronization section.

After the script synchronization is successfully executed, Aquera syncs the Workers created in ADP Workforce Now as users in Google Workspace.
