Overview

The Aquera platform allows your organization to scale complex data management by automating data integration across resource types. Aquera's integrations facilitate the automatic provisioning, ensuring consistency across systems. It allows organizations to streamline processes, dynamically scale resources, and automate repetitive tasks, ultimately improving efficiency in managing intricate IT infrastructures and applications.

This guide outlines the process for setting up an integration to retrieve pre-hire (candidates) data from ADP Workforce Now and generate a CSV report that includes all candidate attributes. The CSV report is then shared with the specified recipients via email.

Prerequisites

The following are the prerequisites:

  • Tenant Account in the Aquera Admin portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure the ADP Workforce Now application connector in the Aquera Admin portal: Refer to the ADP Workforce Now Configuration Guide for details.

Integration Setup

This section outlines the configuration required to send the generated CSV file to the designated users via email and to display the integration results on the dashboard. It covers creating the integration, configuring system preferences, scheduling the integration to run, and distributing the generated CSV to the designated users. 

Create Integration

Perform the following steps to create the integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Select ADP Job Application.
  5. Enter a name and a description for the integration.
  6. Select your ADP Workforce Now connector instance.
  7. Select your CSV Template connector instance.
  8. Click Continue. The Configuration page is displayed to configure the integration.

Configuration

The following tabs are displayed on the Configuration page:

Applications

The Applications tab displays the Applications section, which displays the application connectors for the integration. This section allows you to search and update these application connectors.

Parameters

The Parameters section enables you to customize the integration settings according to your specific use case. It has the following parameters:

  • Fetch the Job Applications by Application Status: Select the application status from the dropdown menu to filter the job applications while retrieving.
  • Enable Integration Report Download: Select to allow downloading the integration report in CSV format.
  • Enable Debug Mode: Select to generate logs outlining data changes that would occur for debugging.
  • Enable Dashboard Generation: Select to analyze integration-related metrics in a dashboard.
  • Send Email Notification: Select to receive integration-related email notifications.
  • Send Integration Report Via Email: Select to receive email notification with a CSV report for all integration runs.
  • Recipients' Email Addresses: Enter the email addresses of the recipients to whom the email notifications will be sent
  • Email Subject: Enter the content that must be used as the email subject
  • Email Body: Enter the content that must be used as the email body for all email notifications

Schedule

Integrations can be scheduled to run automatically based on your choice of timeframe. Refer to Integration Schedule Setup Guide for further details.

Run Integration

You can set up an event or schedule triggers to execute the integration. For further details, refer to the Integration Using Scripts guide.

Once the integration is successfully executed, Aquera sends the CSV report via email to the specified administrators.

Aquera provides additional options in its integration solutions, allowing you to perform various actions on integrations, including executing, upgrading, copying, modifying, deleting, exporting, importing, viewing logs, and downloading reports. Refer to User Actions on Integrations for further details.
