Overview

When setting up an integration on the Aquera platform to synchronize data between applications, you need to configure several settings based on your specific use case.

This guide provides detailed information on all configuration options available within the sections of the Configuration tab for an integration.

  • Connection - Set the source and target applications that must be used in the integration.
  • Onboarding - Select Enable Onboarding to view the employee onboarding configuration fields.
  • Profile Update - Select Enable Profile Update to view the configuration fields related to the user profile updates in the target application.
  • Offboarding - Select Enable Offboarding to view the employee offboarding configuration fields.
  • Additional Settings - Navigate to this section to configure the advanced settings for this integration, which include source filter condition and destination modification safeguard settings
  • Execution - Navigate to this section to configure the integration execution settings, which include import limit and update limit configurations.
  • Report Settings - Navigate to this section to configure the reports generated by the integration.
