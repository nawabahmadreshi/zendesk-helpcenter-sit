Overview

The Aquera platform allows your organization to scale complex data management by automating data integration across resource types. Aquera's integrations facilitate the automatic provisioning, ensuring consistency across systems. It allows organizations to streamline processes, dynamically scale resources, and automate repetitive tasks, ultimately improving efficiency in managing intricate IT infrastructures and applications.

This guide outlines how hiring manager details in iCIMS job records are updated automatically by syncing them with the latest company data from HubSpot Contacts.

To identify company records across both systems, the integration uses the hs_object_id attribute from HubSpot Contacts and matches it to a custom attribute in iCIMS.

Key Functions

The HubSpot to iCIMS Hiring Manager Sync integration provides the following functions:

  • Fetches HubSpot companies based on configured search filters.
  • Retrieves active iCIMS people records by matching email addresses from HubSpot company data.
  • Updates eligible iCIMS jobs with mapped hiring manager details
  • Matches HubSpot companies to iCIMS companies using defined source and target attributes.

Prerequisites

The following are the prerequisites:

  • Aquera Tenant Account: You must have a tenant account in the Aquera Admin portal. Visit https://admin.aquera.io to register for a tenant account. Contact your Aquera account manager to get Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure the following application connectors in the Aquera Admin Portal:
    • HubSpot Contacts: Refer to the HubSpot Contacts Configuration Guide for details.
    • iCIMS: Refer to the iCIMS Configuration Guide for details.

Integration Setup

This section outlines how to configure the integration to push updated job records with new hiring manager data from HubSpot Contacts to iCIMS.

Create Integration

Perform the following steps to create the integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Select HubSpot Contacts to iCIMS Hiring Manager Sync.
  5. Enter a name and a description for the integration.
  6. Select your HubSpot Contacts and iCIMS connector instances.
  7. Click Continue. The Configuration page is displayed to configure the integration.

Configuration

The following sections are displayed under the Configuration tab:

The Applications section displays the source and target application connectors for the integration. This section allows you to search and update these application connectors.

The Parameters section enables you to customize the integration settings according to your specific use case. It has the following settings:

Integration Features

Configure the following integration features:

  • Enable Hiring Manager Sync: Select to enable hiring manager data before running the sync.

Execution Features

Configure the following execution features:

  • Enable Report-Only Mode: Select to generate a report detailing the data changes that would happen, but without applying them
  • Enable Logging: Select to generate logs outlining data changes
  • Enable Notifications: Select to enable the integration-related notifications.
  • Enable Report Generation: Select to generate the integration report.
  • Enable Log Insights: Select to visualize the integration-related metrics.
  • Enable Debug Mode: Select to generate logs outlining data changes that would occur for debugging.
  • Full Sync Validity(Days): Defines how long a full sync remains valid. If the previous full sync is older than the specified number of days, the integration may automatically trigger a new one.
