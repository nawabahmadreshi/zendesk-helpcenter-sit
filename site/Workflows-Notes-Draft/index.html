Workflows

This section provides information on the Workflows module that is newly introduced in the Aquera Admin portal.

New Feature

A new module Workflows is introduced in the Aquera Admin portal that allows you to set up a sequence of tasks to be performed on the data (like users, groups, roles, tickets, or files) based on the resource types supported by the application. The tasks can be any operations such as reading, creating, updating, and deleting entities, and any general tasks such as sending emails, creating service requests, and so on.

You can set a schedule to trigger the workflows to run automatically.

The following options are available to set up the tasks in a workflow:

  • Actions
    • Read
    • Create
    • Update
    • Notification
  • Conditions
    • Yes or No
    • Cases

For further details, refer to the Workflows User Guide
