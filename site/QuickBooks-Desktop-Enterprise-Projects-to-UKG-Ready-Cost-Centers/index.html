QuickBooks Desktop Enterprise Projects to UKG Ready Cost Centers

Overview

The Aquera platform allows your organization to scale complex data management by automating the synchronization of data across resource types. This guide covers the end-to-end tasks in setting up a script synchronization to sync the Projects resource of QuickBooks Desktop Enterprise to the Cost Centers resource of UKG Ready.

Prerequisites

The following are the prerequisites:

  • Admin Access to QuickBooks Desktop Enterprise: You must have admin access to your QuickBooks Desktop Enterprise instance. You can visit the QuickBooks Desktop Enterprise portal to get access to an QuickBooks Desktop Enterprise instance.
  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure following application connectors in the Aquera Admin portal to integrate with QuickBooks Desktop Enterprise:
    • QuickBooks Desktop Enterprise: Refer to the QuickBooks Desktop Enterprise Configuration Guide for details.
    • UKG Ready: Refer to the UKG Ready Configuration Guide for details.

Synchronization Setup

This section details how the synchronization must be configured to allow data push from QuickBooks Desktop Enterprise to UKG Ready.

Create Script Synchronization

Perform the following steps to create a new script synchronization:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Synchronizations from the left side panel.
  3. Click Create Synchronization.
  4. Select QuickBooks Desktop Enterprise Projects to UKG Ready Cost Centers.
  5. Enter a name and a description for the synchronization.
  6. Select your QuickBooks Desktop Enterprise and UKG Ready connector instances.
  7. Click Continue. A new page is displayed to configure the synchronization.

Attribute Mapping

The QuickBooks Desktop Enterprise Projects to UKG Ready Cost Centers template script maps the commonly used QuickBooks Desktop Enterprise and UKG Ready attributes. You can customize it based on your use case.
The following are the attributes that are mapped by default:

QuickBooks Desktop Enterprise Script Expression UKG Ready Script Expression
companyName (customerDetailsWithJobNameAndDescription[i].ParentRef.FullName) ukginstanceCompanyName
jobName (customerDetailsWithJobNameAndDescription[i].Name)  cost_centers.name
customerType (customerDetailsWithJobNameAndDescription[i].CustomerTypeRef.FullName)  cost_centers.description
treeIndex (ROOT_COST_CENTER_FIELD_VALUE - 1) cost_centers.parent_cc. tree_index
companyNameId (companyCostCenter.id) cost_centers.parent_cc.id

Run Script Synchronization

You can set event or schedule triggers to run the script synchronization. For more details, refer to the Trigger section.

After the script synchronization is successfully executed, Aquera syncs the contractors created in UKG Ready as vendors in QuickBooks Desktop Enterprise.
