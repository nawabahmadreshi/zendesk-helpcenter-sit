Greenhouse Onboarding Photos to BambooHR Photos

← Back to index

The Aquera platform allows your organization to scale complex data management by automating the synchronization of data across resource types. This guide covers the end-to-end tasks in setting up a script synchronization to sync the Photos resource of Greenhouse Onboarding to the Photos resource of BambooHR.

Prerequisites

The following are the prerequisites:

  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure following application connectors in the Aquera Admin portal to integrate with BambooHR:
    • Greenhouse Onboarding: Refer to the Greenhouse Onboarding Configuration Guide for details.
    • BambooHR: Refer to the BambooHR Configuration Guide for details.

Synchronization Setup

This section details how the synchronization must be configured to allow data push from Greenhouse Onboarding to BambooHR.

Create Script Synchronization

Perform the following steps to create a new script synchronization:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Synchronizations from the left side panel.
  3. Click Create Synchronization.
  4. Select Greenhouse Onboarding Photos to BambooHR Photos.
  5. Enter a name and a description for the synchronization.
  6. Select your Greenhouse Onboarding and BambooHR instances.
  7. Click Continue. A new page is displayed to configure the synchronization.

Attribute Mapping

The Greenhouse Onboarding Photos to BambooHR Photos template script maps the commonly used Greenhouse Onboarding and BambooHR attributes. You can customize it based on your use case.
The following are the attributes that are mapped by default:

Greenhouse Onboarding Script Expression BambooHR Script Expression
ghEmployee.profileImage base64Photo
ghEmployee.profileImage.fileUrl base64Photo
ghEmployee.email bambooEmployee

Note: bambooEmployee finds the matching employee using the  findUserByUsername method. The full expression is bambooEmployee  = bamboohr.findUserByUsername(ghEmployee.email).

Run Script Synchronization

You can set event or schedule triggers to run the script synchronization. For more details, refer to the Executing a Synchronization section.

After the script synchronization is successfully executed, Aquera syncs the Photos created in Greenhouse Onboarding as Photos in BambooHR.
