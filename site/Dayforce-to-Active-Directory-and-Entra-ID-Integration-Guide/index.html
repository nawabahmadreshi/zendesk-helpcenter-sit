Overview

The Aquera platform enables your organization to scale complex data management by automating data integration across various resource types. Aquera's integrations facilitate automatic provisioning, ensuring consistency across systems. It enables organizations to streamline processes, dynamically scale resources, and automate repetitive tasks, ultimately enhancing efficiency in managing complex IT infrastructures and applications.

This integration synchronizes employee and candidate data from Dayforce to Active Directory and Microsoft Entra ID to automate the full employee lifecycle. It provisions new user accounts for onboarding, updates user attributes when employee information changes, and deprovisions or disables users upon termination based on Dayforce status. It enables centralized identity management and supports automated application access and entitlement assignment using predefined rules.

Key Functions

This integration automates identity lifecycle management with the following primary capabilities:

  • Synchronizes employee and candidate data from Dayforce to Active Directory and Microsoft Entra ID.
  • Manages group assignments.
  • Generates password for new users in Active Directory and Microsoft Entra ID.
  • Sends notifications about provisioning events.
  • Generates detailed logs and reports.

Prerequisites

The following are the prerequisites:

  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/.
  • User Account in Entra ID: You must have a user account in Entra ID with permission to configure provisioning (for example, Application Administrator, Cloud Application Administrator, Application Owner, or Global Administrator).
  • Application Connectors in the Aquera Admin Portal: You need to configure the following application connectors in the Aquera Admin Portal:
    • Dayforce: Refer to the Dayforce Configuration Guide for configuration details.
    • Active Directory: Refer to the Active Directory (AD) Configuration Guide for configuration details.
    • Microsoft 365: Refer to the Microsoft 365 Configuration Guide for configuration details.

Integration Setup

This section details the configuration and mapping process for integrating Dayforce with Active Directory and Microsoft Entra ID. It outlines the steps to create the integration, set up integration preferences, and define how data is mapped.

Create Integration

Perform the following steps to create the integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Search for Dayforce to Active Directory / Entra ID in the search box next to the Explore Templates section, and then select it to create a new integration.
  5. Enter a name and a description for the integration.
  6. Click Continue.

Once created, the following tabs are displayed:

  • Overview
  • Source
  • Directory
  • Policy
  • Execution
  • Notification Center
  • Tables
  • Schedule

Overview

On this tab, you can view integration details, including configuration settings, and access the available features for enablement.

Scroll to the Features List section to enable or disable integration features based on your use case.

Features List

This section lists the features supported by the integration. You can enable or disable each feature based on your requirements.

The following are the supported features:

  • Enable Entra ID for Employee Management: Enable this option to synchronize employee data with Microsoft Entra ID. When enabled, the integration provisions new user accounts, updates user attributes based on changes, and deprovisions accounts according to employee lifecycle events and data changes in Dayforce.
  • Enable Entra ID for Candidate Management: Enable this option to synchronize candidate data with Microsoft Entra ID. When enabled, candidate accounts can be provisioned and managed in Microsoft Entra ID.
  • Enable Active Directory for Employee Management: Enable this option to synchronize employee data with Active Directory. When enabled, the integration creates, updates, and deprovisions user accounts in Active Directory based on employee lifecycle events and data changes in Dayforce.
  • Enable Active Directory for Candidate Management: Enable this option to synchronize candidate data with Active Directory. When enabled, candidate accounts can be provisioned and managed in Active Directory.
