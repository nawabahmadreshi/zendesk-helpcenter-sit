Overview

The Aquera platform allows your organization to scale complex data management by automating the synchronization of data across resource types. This guide covers the end-to-end tasks in setting up a script synchronization to sync the contractors from the Candidates resource of UKG Ready to the Contacts resource of Salesforce.

Prerequisites

The following are the prerequisites:

  • Admin Access to Salesforce: You must have admin access to your Salesforce instance.
  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure following application connectors in the Aquera Admin portal to integrate with Salesforce:
    • UKG Ready: Refer to the UKG Ready Configuration Guide for details.
    • Salesforce: Refer to the Salesforce Configuration Guide for details.

Synchronization Setup

This section details how the synchronization must be configured to allow data push from UKG Ready to Salesforce.

Create Script Synchronization

Perform the following steps to create a new script synchronization:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Synchronizations from the left side panel.
  3. Click Create Synchronization.
  4. Select UKG Ready Candidates to Salesforce Contacts.
  5. Enter a name and a description for the synchronization.
  6. Select your UKG Ready and Salesforce connector instances.
  7. Click Continue. A new page is displayed to configure the synchronization.

Attribute Mapping

The UKG Ready Contractors to Salesforce Vendors template script maps the commonly used UKG Ready and Salesforce attributes. You can customize it based on your use case.
The following are the attributes that are mapped by default:

UKG Ready Script Expression Salesforce Script Expression
Candidate.id Contact.UKG_ID__c
Candidate.employee_id Contact.UKG_Employee_ID__c
Candidate.first_name Contact.FirstName
Candidate.last_name Contact.LastName
Candidate.primary_email Contact.HPG_Emailc / Contact.Email
Candidate.address.address_line_1 + Candidate.address_line_2 + Candidate.address_line_3 Contact.MailingStreet
Candidate.address.country Contact.MailingCountry
Candidate.address.city Contact.MailingCity
Candidate.address.state Contact.MailingState
Candidate.address.zip Contact.MailingPostalCode
Candidate.phones.cell_phone Contact.MobilePhone
Candidate.dates.hired Contact.Hire_Date__c
Candidate.dates.review Contact.Review_Date__c
Candidate.dates.terminated Contact.Separation_Date__c
Candidate.dates.birthday Contact.Birthdate
Candidate.dates.birthday (month) Contact.Birth_Month__c
Candidate.dates.birthday (day) Contact.Birth_Date__c
Candidate.default_job.name Contact.HPG_Title__c

Run Script Synchronization

You can set event or schedule triggers to run the script synchronization. For more details, refer to the Trigger section.

After the script synchronization is successfully executed, Aquera syncs the candidates created in UKG Ready as contacts in Salesforce.
