Overview

The Aquera platform allows your organization to scale complex data management by automating data integration across resource types. Aquera's integrations facilitate the automatic provisioning, ensuring consistency across systems. It allows organizations to streamline processes, dynamically scale resources, and automate repetitive tasks, ultimately improving efficiency in managing intricate IT infrastructures and applications.

The ADP Workforce Now to SQL server integration extracts the workers, payroll, department, and personal contacts resource types and their selected attributes from the ADP Workforce Now - API Central app and creates the tables in the configured SQL server database. 

Refer to the ADP Workforce Now / Microsoft SQL Server Integration Template Version History to view new features and improvements added in different versions of this integration template.

Prerequisites

The following are the prerequisites:

  • Aquera Tenant Account: You must have a tenant account in the Aquera Admin portal. Visit https://admin.aquera.io to register for a tenant account. Contact your Aquera account manager to get Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure the following application connectors in the Aquera Admin portal for integration:
    • ADP Workforce Now: Refer to the ADP Workforce Now Configuration Guide for ADP API Central Customers for details.
    • Microsoft SQL Server: Refer to the Microsoft SQL Server Configuration Guide for details.
