Overview

The Aquera platform allows your organization to scale complex data management by automating the integration of data across resource types. This guide covers the end-to-end tasks in setting up a script integration to sync the Candidates resource of iCIMS to the Workers resource of ADP Workforce Now API Central.

Prerequisites

The following are the prerequisites:

  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure following application connectors in the Aquera Admin portal to integrate with Microsoft Entra ID:
    • iCIMS: Refer to the iCIMS Configuration Guide for details.
    • ADP Workforce Now: Refer to the ADP Workforce Now Configuration Guide for ADP API Central Customers for details.

Integration Setup

This section details how the integration must be configured to allow data push from iCIMS to ADP Workforce Now.

Create Script Integration

Perform the following steps to create a new script integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to integrations from the left side panel.
  3. Click Create integration.
  4. Select iCIMS Candidates to ADP Workforce Now Workers.
  5. Enter a name and a description for the integration.
  6. Select your iCIMS and ADP Workforce Now API Central connector instances.
  7. Click Continue. A new page is displayed to configure the integration.

Run Script Integration

You can set event or schedule triggers to run the script integration. For more details, refer to the Trigger section of the Integration Using Scripts guide.

After the script integration is successfully executed, Aquera syncs the candidates created in iCIMS as workers in ADP Workforce Now.
