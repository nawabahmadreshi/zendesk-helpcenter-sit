Overview

American Express @ Work is a business platform that helps organizations manage corporate cards, expenses, and payments efficiently. It provides tools for expense tracking, reporting, and account management to support better financial control and visibility.

This guide helps users to create an American Express @ Work connector on the Aquera platform.

For further details on the American Express @ Work application, visit the American Express @ Work website.

Supported Features

The American Express @ Work connector supports various features as per the OpenAPI specification. Refer to the OpenAPI Specification section to know all supported OpenAPI operations.

Prerequisite

You must have the admin credentials of your American Express @ Work instance.

Connector Configuration

Perform the following steps to configure the American Express @ Work connector:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Applications and click Add Application.
  3. Search for American Express @ Work and click the American Express @ Work icon.
  4. Enter the values for the following fields under the Configuration tab:
    • Basic Details:
      • Display Name: Enter a display name for your application.
      • Description: Enter a description for your application.
      • Tenant: Enter the hostname available in the login URL of your American Express @ Work instance. For example, if https://aquera.abc.com/ is the login URL, configure the tenant name as aquera.abc.com.
      • Source Owner: Enter valid email IDs of the source owners.
      • Subscribe to error notification: Enable the toggle button to send notification emails to source owners for errors in the user data transactions. Click Edit Preference to select the error types for which you want to receive notifications. After you configure your preference, you will receive error notifications only for the selected error types. By default, all error types are selected.
      • Pick your protocol: Select the protocol from the dropdown list based on your requirements. SCIM is the default value.
      • Copy this URL: This field displays the base URL of the connector API endpoint. The URL is populated based on your selected protocol. Copy and paste this URL while provisioning to the identity service provider or performing OpenAPI operations, if supported.
  5. Click Create to create the American Express @ Work connector.

Refer to the Getting Started with Connectors guide to learn more about connector setup including different tabs on the connector configuration page.

Authentication

The American Express @ Work connector supports Basic Authentication. Provide the username and password of the American Express @ Work administrator account for authentication.
