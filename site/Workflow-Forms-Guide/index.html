Overview

This guide details how to use forms in workflows. Aquera's Forms feature offers the option to gather inputs from a set of recipients. Built into workflows, Forms also have the capability to trigger other workflows.

Creating Forms

Perform the following steps to create a form in the Aquera Admin portal:

  1. Sign in to the Aquera Admin portal.
  2. Navigate to the Workflows module and click Create Workflow at the top-right corner of the page.
  3. Click Blank to create a new workflow. 
  4. Navigate to the Invoke section under the Components panel and drag & drop Form into the workflow creation page.

Configuring Forms

Perform the following steps to configure a form in the Aquera Admin portal:

  1. Click the form you just added to the workflow creation page to open the form configuration panel.

  2. Enter the email addresses of recipients under Recipient email ids and hit the Enter key.

  3. Click Create Form to open the form creation sheet. For more information, refer to the Generating Form Sheet section.

  4. After generating the form sheet, you can invoke workflows upon clicking the buttons added to the form. Pick the desired workflow from the dropdown list to initialize it when clicking the respective button while filling out the form.

Generating Form Sheet

Perform the following steps to generate a form sheet in the Aquera Admin portal:

  1. Enter a name and description for the form in the first form block.  
  2. Enter a question or prompt in the second form block and choose the appropriate input method. The available options are Multi Select, Single Select, Dropdown, Short Text, MultiLine Text, and No Input.
  3. Enter the answer options based on the input method you have chosen. Click the Add more options button to include additional options to the answer.

  4. Enable the Mandatory toggle to set a question as mandatory. You also have the option to click the trash icon to delete a form block.
  5. Click the + icon to add new blocks to the form creation page. You can then configure the newly added form block to add more questions to the form.
  6. Click the Untitled Button button to rename it. You can click the Add more button option to include additional buttons in the form.

Customizing Form Sheet

Perform the following steps to customize the form sheet:

  1. Click the color palette icon to change the form's background color.
  2. Click the Upload icon to set the logo and cover images.

    Note: The maximum file size of the logo and cover images must be under 1MB and 2MB respectively.
  3. Crop the portion you want in the logo or cover and click Done to set the logo/cover image.
  4. Pick one of the available colors to change the background color of the form.

Preview and Save the Form

Perform the following steps to preview and save forms:

  1. Click the eye icon at the top-right corner of the form creation page to preview the form.
  2. Click Save Form to save the changes.

Filling Forms

Perform the following steps to fill and submit forms:

  1. Open the form's link from the email inbox of the address you entered in the Recipient email ids field. 
  2. Fill in the details and click the button to submit the form.

Note: After clicking the button, the workflow mapped to the button through the Action Mapping option will get triggered. If you have multiple buttons in the form, you can click any of the buttons to submit the data. To change the workflow trigger action for buttons, refer to the Configuring Forms section.
