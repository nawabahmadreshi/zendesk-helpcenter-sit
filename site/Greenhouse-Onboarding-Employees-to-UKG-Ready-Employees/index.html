Greenhouse Onboarding Employees to UKG Ready Employees

Overview

Aquera provides an innovative platform that allows your organization to scale complex data management by automating the integration of employees from the Greenhouse Onboarding application to the employees in the UKG Ready application.

This guide covers the end-to-end tasks in setting up integration to synchronize Greenhouse Onboarding employees to UKG Ready employees. 

Prerequisites

The following are the prerequisites:

  • Aquera Tenant Account: You must have a tenant account in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.

    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/

  • Configure Applications in the Aquera Admin Portal: Aquera provides SCIM Gateway connectors to support employee lifecycle management. Aquera's connectors use the SCIM protocol to perform the read and write operations on the applications. You must add and configure the following connectors in the Aquera Admin portal:
    • Greenhouse Onboarding: Refer to the Greenhouse Onboarding Configuration Guide for details.
    • UKG Ready: Refer to the UKG Ready Configuration Guide for details.

Data Integration Setup

Aquera's data integration automatically imports employees from the Greenhouse Onboarding application and provisions these employees in the UKG Ready application.

Aquera provides templates that are pre-configured integrations created for specific use cases. You can use these templates to create a data integration.

Perform the following steps to create a data integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations on the left panel.
  3. Click Create Integration.
  4. Select a pre-built data integration template based on your requirements. For this use case, select the template Greenhouse Onboarding to UKG Ready Employee.
  5. Enter a name and a description for the integration.
  6. Select a Greenhouse Onboarding connector in the Source Application dropdown field.
  7. Select a UKG Ready connector in the Target Application dropdown field.
  8. Click Continue. A new page with the fields and tabs to configure the data integration is displayed.
  9. You can update the configurations if required. For further details, refer to the Data Integration Setup Guide. 
  10. Click Create to create the data integration.

Attribute Mapping

The Attribute Map tab allows you to map employee attributes from the Greenhouse Onboarding application to the employee attributes in the UKG Ready application. 

The following is a sample attribute mapping from Greenhouse Onboarding to UKG Ready:

Greenhouse Onboarding Attribute / Value / Expression UKG Ready Attribute
SOURCE.customFieldValues__Employee_Number.value employeeId*
SOURCE.firstName firstName*
SOURCE.lastName lastName*
SOURCE.middleName middleName
SOURCE.suffix suffix
SOURCE.userName username*
SOURCE.startDate dates.started*
SOURCE.customFieldValues__Hire_Date.value dates.hired*

In the source attribute field, you can include source attributes or write JavaScript expressions to map to the corresponding target attribute. The target attributes marked with a red asterisk (*) are mandatory to be mapped.

Running the Data Integration

You can click the Execute button at the top right of the data integration to manually run the data Integration. You can also schedule the data integration to run automatically at specified intervals by configuring a frequency in the Schedule tab.

You can navigate to Logs  > Integrations in the Aquera Admin portal to view the logs of the integration execution.

After the data integration is successfully executed, new employees will be created and existing employees will be updated in the UKG Ready application.
