Overview

Aquera provides an innovative platform that allows your organization to scale complex data management by automating the synchronization of employees from the UKG Ready application to the employees in the Acumatica application.

This guide covers the end-to-end tasks in setting up synchronization to synchronize UKG Ready employees to Acumatica employees. 

Prerequisite

Tenant account in the Aquera Admin portal: You must have a tenant account in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.

Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/

Configure Applications in the Aquera Admin Portal

Aquera provides SCIM Gateway connectors to support employee lifecycle management. Aquera's connectors use the SCIM protocol to perform the read and write operations on the applications. 

You must add and configure the following connectors in the Aquera Admin portal:

  • UKG Ready: Refer to the UKG Ready Configuration Guide for details.
  • Acumatica: Refer to the Acumatica Configuration Guide for details.

Synchronization Setup

Aquera's Synchronization automatically synchronizes data from a source to the target application.
