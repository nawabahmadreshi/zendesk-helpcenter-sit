Prerequisites

The following are the prerequisites:

  • Admin Access to Zendesk: You must have admin access to your Zendesk instance. You can visit the Zendesk portal to get access to Zendesk instance.
  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/.
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure the following application connectors in the Aquera Admin portal to integrate with Zendesk:
    • Entra ID: Ensure that you configure the Entra ID connector in the Aquera Admin portal. Refer to the Microsoft 365 Configuration Guide for configuration details.
    • Zendesk: Ensure that you configure the Zendesk connector in the Aquera Admin portal. Refer to the Zendesk Configuration Guide for configuration details.

Integration Setup

This section details the configuration and mapping process for integrating Entra ID / Zendesk. It outlines the steps to create the integration, set up integration preferences, and define how candidate data is mapped from Entra ID / Zendesk.

Create Integration

Perform the following steps to create the integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Search for Entra ID / Zendesk in the search box next to the Explore Templates section, and then select it to create a new integration.
  5. Enter a name and a description for the integration.
  6. Click Continue. The Configuration page is displayed to configure the integration.
