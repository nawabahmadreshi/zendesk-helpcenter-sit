The Aquera platform allows your organization to scale complex data management by automating the synchronization of data across resource types. This guide covers the end-to-end tasks in setting up a script synchronization? to sync the Users resource of Active Directory to the Employees resource of ADP Workforce Now.

Prerequisites

The following are the prerequisites:

  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure following application connectors in the Aquera Admin portal to integrate with BambooHR:
    • Active Directory: Refer to the Active Directory (AD) Configuration Guide for details.
    • ADP Workforce Now: Refer to the ADP Workforce Now Configuration Guide for details.
