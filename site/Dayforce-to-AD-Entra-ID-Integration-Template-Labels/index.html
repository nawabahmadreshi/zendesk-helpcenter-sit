Overview

About this Integration

This integration automates user provisioning between Dayforce, Active Directory, and Microsoft Entra ID, syncing employee and candidate data, managing group assignments, and handling onboarding, updates, and terminations. It also supports automated access assignments with detailed notifications, logs, and reports.

Execution Configuration

Schedule

The integration supports automatic execution based on the configured schedule.

Notification Centre

The integration supports email notifications for various provisioning events.

Operational Settings

The integration supports the following:

  • Customized logging
  • Customized reporting
  • Customized dashboards
  • Customized operational settings for different target applications.
