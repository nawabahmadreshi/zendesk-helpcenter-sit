Zendesk Setup Guide:
1. Go to Admin Center u2192 Apps and integrations u2192 Webhooks.
2. Create a webhook for Guide u2192 Articles with POST method.
3. Use the URL above and JSON format.
4. Create a Trigger in Guide to notify this webhook on article publish/update.

OpenRouter Cheap Cloud
Access 100+ models with one key. Many are free. Connects to Mistral, Llama-3, Phi-3, Claude, GPT-4 and more via a single OpenAI-compatible API. openrouter.ai →
Ollama Local daemon
Ollama runs locally as a separate service. Install it once, then run any open-source model. Mac: brew install ollama or download from ollama.ai.
Linux: curl -fsSL https://ollama.ai/install.sh | sh
Docker (sidecar): See the docker-compose.yml in the project root — Ollama runs as a container alongside the app.
Pull a model first: ollama pull phi3:mini
Embedded Local AI Zero cost
Uses llama-cpp-python — runs a GGUF model entirely inside this Python process. No external service needed. Requires ~2.5 GB disk, uses GPU (Metal/CUDA) if available.

Cost comparison: Gemini Flash (~$0.15/1M tok) → OpenRouter free models ($0) → Ollama/Local ($0, self-hosted CPU/GPU).
Auto mode tries Gemini first (fast + smart), falls back to OpenRouter or local if it fails — best of both worlds.
