BambooHR Webhook Script Integration Guide

← Back to index

Overview

This guide covers information on how the BambooHR Webhook Script integration can be set up in the Aquera Admin console. It also details the webhook setup to be done in the BambooHR system to send user events to Aquera.

Once the setup is complete, all user events sent from the BambooHR system are automatically handled by Aquera and processed for user provisioning purposes.

The following procedures are involved in the Webhook setup:

  • Setting up BambooHR Webhook Script Integration
  • Setting Up Webhooks in BambooHR

Setting up BambooHR Webhook Script Integration

NOTE:

 
  If Webhooks is not enabled in your BambooHR account, you must send an email to support@bamboohr.com to enable Webhooks.
  1. Log in to the Aquera Admin console, https://admin.aquera.io/.
  2. Click Integrations on the left navigation pane and click Create Integration. The Add Integration page is displayed.
  3. Provide a name and description for the integration in the corresponding fields and click Continue.
  4. Select the integration type BambooHR Webhook Script from the integration type drop-down list.
  5. Click Done. The following fields are displayed on the configuration page:
    • Select the target Application to which the user data needs to be sent after processing the events received from BambooHR.
    • Enter the JavaScript to be executed for processing the events received from BambooHR.
      Click the Expand icon, if necessary, to expand the window for entering the Javascript. Here is a sample script to process the inbound event from BambooHR:
       
      log.info(JSON.stringify(event, null, 4));
  6. Click Create. A confirmation message is displayed indicating that the integration has been created successfully.
  7. Click the Event Detail tab. The event URL is displayed.

  8. Copy the URL. This URL has to be configured in the Add Webhook page in BambooHR. Refer to the Setting Up Webhooks in BambooHR section for details.
  9. Click Create Token. The Custom Token field is displayed. Paste the Private Signature Key generated in the BambooHR application in this field.
  10. Use the Private Signature Key generated in the BambooHR application as the Custom token and then click Save Changes. For details on generating the Private Signature Key, refer to the Setting Up Webhooks in BambooHR section.
    The BambooHR Webhook Script integration setup is completed in Aquera Admin.

Setting Up Webhooks in BambooHR

Webhooks must be set up in BambooHR to monitor any changes made to the fields that you specify and to post that information to Aquera. Aquera's BambooHR webhook script will automatically process those changes and send them to the target application, by executing the scripts configured in the integration setup.

  1. Log in to your BambooHR account as an administrator. The Home page is displayed.

  2. Click the Settings icon at the top-right corner of the page. The Settings page is displayed.
  3. Click Webhooks. The Webhooks section is displayed on the page.

  4. Click Add Webhook. The Add Webhook page is displayed.

  5. Enter a name for the webhook in the Webhook Name field.
  6. Select the fields to be monitored by the BambooHR system for any changes, from the What fields do you want to monitor? drop-down list. The fields have to be selected one by one. All the selected fields are listed on the page.

    You can remove any selected fields by clicking the close icon next to the field.

  7. Select the same set of fields in the What fields do you want to post? section.
    • Search for the required field by typing the field name  and selecting it,

      or

    • Browse to the individual categories from the Browse Fields list and then add the required field to the Selected Fields list.

  8. Select JSON in the What format should the data be in? section.

  9. In the Post to URL field, paste the URL that has been copied from the Event tab of the BambooHR Webhook script integration in Aquera Admin. Refer to the Setting up BambooHR Webhook Script Integration section for information on the URL.

  10. Click Generate Key. Copy the Private Signature Key and use it in the Custom Token field in the Event tab on the Aquera Admin portal.
  11. Click Save. The newly created webhook is displayed in the Webhooks section on the Settings page.

    Now, when the specified data of users in the People section of BambooHR is updated, it will be sent to Aquera for processing and sending to the target application.
