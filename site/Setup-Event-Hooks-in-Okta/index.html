Overview

Aquera integrations can be triggered when specific events occur in an application. An event notification will be sent from the application when data changes in the application, such as the creation of a new user, modification of an existing user, activation or deactivation of a user, addition of a user to a group, or removal of a user from a group. We need to configure event hooks in the application to receive these events.

To receive event notifications from Okta, subscribe to the events in the Okta application. Once subscribed, Okta will send notifications whenever the specified events occur for user accounts. These are HTTPS REST calls containing event data in JSON format and are sent to the event URL configured in the integration on the Aquera Admin portal. 

Aquera integration then processes these events and updates the corresponding data in a target application.

For details on how to set up an integration in the Aquera platform, refer to the Integration Using Scripts guide.

This guide explains the procedure to set up event hooks in Okta.

Set up Event Hooks in Okta

Perform the following steps to set up the event hooks in Okta:

  1. Log in to your Okta administrative console.
  2. Navigate to Workflow > Event Hooks. The Event Hooks page is displayed.
  3. Click Create Event Hook. The Create Event Hook page is displayed.
  4. Enter values in the following fields:
    Field Description
    Endpoint URL Enter the external service's endpoint URL where the event hook sends the request.
    Event Hook name Enter a name for the event hook.
    Description Enter the description for the event hook.
    Customize Requests
    Authentication field

    Enter the name of the authorization header.

    Note: Okta Event Hooks uses header-based authentication.
    Authentication secret

    Enter the value that corresponds to the field name.
    Custom header fields

    Optional field names and value pairs to be sent with the request to Aquera integration.

    Select Events
    Select all events that apply

    Select all events that must be applied to the Aquera integration.
  5. Click Create hook & Continue.
  6. You can apply filters to events, if required.
  7. Click Next. The Preview page is displayed, allowing you to review the event hook settings.
  8. To skip this step, click Skip this step. Alternatively, you can select an Event Type from the dropdown menu. If you select an event type, the Next button will be available for you to proceed.
  9. Click Verify to activate the hook.
