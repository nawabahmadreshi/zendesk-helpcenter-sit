Integration Setup

This section outlines how the integration must be configured to push worker data from ADP Recruiting Management to ManageEngine ServiceDesk.

Create Integration

Perform the following steps to create the integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Search for ADP Recruiting Management / ManageEngine ServiceDesk integration.
  5. Enter a name and a description for the integration.
  6. Select both application connector instances from the dropdown menus.
  7. Click Continue. The integration configuration page is displayed to configure the integration.

Configuration

The following sections are displayed under the Configuration tab:

Applications

The Applications section displays the source and target application connectors for the integration. This section allows you to search and update these application connectors.

Parameters

The Parameters section enables you to customize the integration settings according to your specific use case. It has the following settings:

Integration Scenarios

Configure the following integration scenarios:

  • Enable worker sync: Select this checkbox to sync worker data from ADP Lyric HCM to ABC Fitness. When enabled, the Worker Configuration section is displayed for configuring additional integration settings.

General Settings

Configure the following general settings:

  • Enable debug mode: Select this checkbox to generate logs outlining data changes that would occur for debugging. When enabled, the Debug Mode section is displayed for further configuration.
  • Enable report-only mode: Select this checkbox to execute the integration in the report-only mode. The report-only mode generates a report to show the data that would be changed without actually making those changes.
  • Enable logging: Select this checkbox to generate logs outlining data changes. When enabled, the Log Settings section is displayed for further configuration.
  • Enable report generation: Select this checkbox to generate the integration report. When enabled, the Report Settings section is displayed for further configuration.
  • Enable notifications: Select this checkbox to enable integration-related notifications. When enabled, the Notification Settings section is displayed for further configuration.
  • Enable log insights: Select this checkbox to visualize the integration-related metrics. When enabled, the Log Insights Settings section is displayed for further configuration.
  • Reset sync state: Select this checkbox to reset the current synchronization state and restart the sync from the beginning.
  • Enable full sync: Select this checkbox to execute the integration in full sync. If this checkbox is not selected, the integration will be executed in the incremental import mode, syncing only new and updated worker records from ADP Lyric HCM to ABC Fitness.
    • Full sync interval(days): Enter the number of days after which a full sync should run. For example, enter 7 to run a full sync every 7 days.

Worker Configuration

The Worker Configuration section enables you to customize the integration settings according to your specific use case. It has the following settings:

Note: This section appears only if Enable worker sync is enabled in the Integration Scenarios section.

Worker Provision Settings

Configure the following worker provision settings:

  • ADP Lyric HCM correlation attribute: Select the ADP Lyric HCM correlation attribute to distinctly find the associated ABC Fitness worker. For example, associateOID.
  • ABC Fitness correlation attribute: Select the ABC Fitness correlation attribute to distinctly find the associated ADP Lyric HCM worker. For example, barcode.
  • Create worker: Select this checkbox to create a worker from ADP Lyric HCM to ABC Fitness. When enabled, the Create Worker Settings section is displayed for further configuration.

Create Worker Settings

Configure the following create worker settings:

  • Create Condition: Select a condition from the dropdown menu to determine when workers are created. The following are the available options:
    • All workers: Select this option to create all workers from ADP Lyric HCM in ABC Fitness.
    • All active workers: Select this option to create all active workers from ADP Lyric HCM in ABC Fitness.
    • All active full-time workers: Select this option to create all active full-time workers from ADP Lyric HCM in ABC Fitness
    • Custom: Select this option to define a custom condition to create a worker from ADP Lyric HCM in ABC Fitness. When enabled, the following field is displayed:
      • Custom create condition: Enter the JavaScript expression to create a worker from ADP Lyric HCM in ABC Fitness.
  • Create limit: Specify the maximum number of worker records that can be created in the target application during a single integration run.

Club Number Settings

Configure the following club number settings:

  • Tenant ID: Enter the unique identifier to retrieve the list of associated club numbers.
  • Club numbers: Select one or more club numbers from the dropdown list to be synced.

Attribute Map

The Attribute Map tab defines how data is mapped from the ADP Lyric HCM to the ABC Fitness application. The default mapping of attributes is displayed on the page. The source attributes are listed on the left side, and the target attributes are listed on the right side of the page.

You can modify only the source attributes as needed to align with your specific use case.

You can write JavaScript expressions to modify the value of the source attribute and construct a value to be updated in the target attribute. Write the expression directly in the source attribute field or click the Expand icon in the source attribute field to write a long expression.

All attributes marked with a red asterisk (*) indicate required fields for successful provisioning.

After the source-to-target attribute mapping is completed, click Save Changes at the top-right corner of the page to save the integration.
