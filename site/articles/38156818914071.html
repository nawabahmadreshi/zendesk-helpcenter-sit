Obtaining Client ID and Client Secret for isolved HCM Connector

← Back

 
Note: This guide is for INTERNAL REFERENCE only and must not be circulated to customers.

Overview

The isolved HCM connector supports basic authentication. A client ID and a client secret are required as username and password for authentication. When a customer purchases the connector, Hanumanthu Indrakanti (hanu.indrakanti@aquera.com) or Mahalakshmi Baskaran (maha.baskaran@aquera.com) will raise an API request to get the credentials for the customer.

API Request Process

Perform the following steps to raise the request:

  1. Log in to your PartnerHub account.
  2. Click Sales.
  3. Select the API Questionnaire option in the API/Integrations dropdown menu.
  4. In the form, look for a section that asks for the API Integration Type, and then select the New API Integration option.
  5. Complete the questionnaire form and submit the API Request.

The isolved team will review the request and provide the required credentials.
