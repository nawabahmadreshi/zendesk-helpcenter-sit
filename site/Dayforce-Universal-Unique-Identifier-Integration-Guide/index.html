Overview

The Aquera platform enables your organization to scale complex data management by automating data integration across various resource types. Aquera's integrations facilitate the automatic provisioning, ensuring consistency across systems. It enables organizations to streamline processes, dynamically scale resources, and automate repetitive tasks, ultimately enhancing efficiency in managing complex IT infrastructures and applications.

This guide covers the end-to-end process of configuring an Aquera integration to generate and assign a Universal Unique Identifier value to all the users in Dayforce.

Prerequisites

The following are the prerequisites:

  • Admin Access to Dayforce: You must have admin access to your Dayforce instance. You can visit the Dayforce portal to get access to a Dayforce instance.
  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/.
  • Configure Application Connector in the Aquera Admin Portal: You need to configure the following application connector in the Aquera Admin portal to integrate with Dayforce:
    • Dayforce: Ensure that you configure the Dayforce connector in the Aquera Admin portal. Refer to the Dayforce Configuration Guide for configuration details.

Integration Setup

This section outlines how the integration must be configured to assign a Universal Unique Identifier to the users in Dayforce.

Create Integration

Perform the following steps to import an integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Select Dayforce.
  5. Enter a name and a description for the integration.
  6. Select your Dayforce connector instance.
  7. Click Continue. The Configuration page is displayed to configure the integration.

Configuration

The following sections are displayed under the Configuration tab:

Applications

The Applications section displays the Dayforce application connector for the integration. This section allows you to search and update these source connector.

Parameters

The Parameters section enables you to customize the integration settings according to your specific use case. It has the following settings:

  • Select an attribute in Dayforce for Universal Unique Identifier (UUID): Select an attribute in Dayforce that will contain the Universal Unique Identifier generated by the Aquera Integration.
  • Employee Number: Enter the XRefCode used to fetch the corresponding employee record from the system.

Schedule

Integrations can be scheduled to run automatically based on your choice of timeframe. Refer to Integration Schedule Setup Guide for further details.
