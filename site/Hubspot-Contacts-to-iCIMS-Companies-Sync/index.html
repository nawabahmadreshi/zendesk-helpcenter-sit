Create Integration

Perform the following steps to create the integration:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations from the left side panel.
  3. Click Create Integration.
  4. Select HubSpot Contacts to iCIMS.
  5. Enter a name and a description for the integration.
  6. Select your HubSpot Contacts and iCIMS connector instances.
  7. Click Continue. The Configuration page is displayed to configure the integration.

Configuration

The following sections are displayed under the Configuration tab:

Applications

The Applications section displays the source and target application connectors for the integration. This section allows you to search and update these application connectors.

Parameters

The Parameters section enables you to customize the integration settings according to your specific use case. It has the following settings:

Global Settings

Configure the following global settings:

  • Generate CSV Report Enable Report Generation: Select to generate the integration report.
  • Report-Only Mode Enable Report-Only Mode: Select to generate a report detailing the data changes that would happen, but without applying them. 
  • Send Email Notification Send Integration Report via Email: Select to receive email notification with a report for all integration runs.
  • Search for Operations-Created Companies: Select to enable a special search mode that includes companies created or tagged by the Operations team.
  • Source Attribute Mapping List:  Enter the list of source attributes that should be dynamically mapped to the target.
  • Target Fields to Compare and Map Against: Enter the list of target fields that should be dynamically compared and mapped to the source attributes.
  • Notification Recipients: Enter the email addresses of the recipients to whom the email notifications will be sent.
  • Correlation Attribute Pair: Enter the pair of attributes used to uniquely identify and match records between HubSpot and iCIMS for syncing.
