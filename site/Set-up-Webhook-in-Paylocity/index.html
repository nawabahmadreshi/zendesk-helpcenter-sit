Set up Webhook in Paylocity

← Back to index

Overview

Aquera integrations can be triggered when specific events occur in an application. An event notification will be sent from the application when data changes in the application, such as the creation of a new user, modification of an existing user, activation or deactivation of a user, addition of a user to a group, or removal of a user from a group. We need to configure webhooks in the application to receive these notifications.

This guide explains how to generate the event integration URL and token in the Aquera portal. These details are required for Paylocity to enable webhook access for your instances. After generating the URL and token in Aquera, share them with the Paylocity support team. Once the Paylocity team enables webhook access, you will be able to view the webhook logs in the Paylocity UI.

Aquera integration then processes these events and updates the corresponding data in a target application.

For details on how to set up an integration in the Aquera platform, refer to the Integration Using Scripts guide.

Setting up a Webhook

Perform the following steps to set up the webhook:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Integrations on the left panel, which displays the list of integrations created by the user. 
  3. Click Create Integration.
  4. Select Integration.
  5. Enter a name and a description for the integration.
  6. Click Continue. A new page is displayed with the fields and tabs to configure the integration.
  7. Under the Configuration tab, click the + next to Applications. The Add Application pop-up window is displayed. Enter the following details:
    • Application: Select your connector from the drop-down menu.
    • Variable: Enter the variable name.
    • Display Name: Enter the display name.
    • Protocol: Select the protocol from the drop-down menu.
  8. Click Create to create the integration.
  9. Click + next to Trigger, the Add Trigger pop-up window is displayed. Enter the following details:
    • Type: Select Event. 
    • Event Source: Select your connector from the drop-down menu
  10. Click Create Token.
  11. Copy the URL and the generated token, and save them securely. This URL and token must be given to the Paylocity team to enable Webhook.

  12. Contact the Paylocity Support Team to get access to the Paylocity APIs. You will receive the Paylocity Web Services Access Request Form (a sample request form is attached with the basic details) from the Paylocity support team, which you need to fill up and send back to them. In this form, you must enter the following details:

    • Events
      • New Hire: Paste the URL copied in the previous step.
      • Username: Enter the Company name.
      • Password: Paste the token copied in the previous step.

For any clarifications, you can contact the Paylocity Support team on this website.

Note: Contact the Paylocity support team to enable the webhook, as it cannot be configured manually.

Webhook Logs

This section describes how to see webhook logs in the Paylocity UI.

Perform the following steps to see the logs:

  1. Log in to your Paylocity instance using the admin credentials.
  2. Click on the Hamburger icon at the top left corner, the navigation menu will be displayed.
  3. Select Web Link.
  4. Select Web Services > Webhook Log.
  5. The Webhook Log page is displayed.
