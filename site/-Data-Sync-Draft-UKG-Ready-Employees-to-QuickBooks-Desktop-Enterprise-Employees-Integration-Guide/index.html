Overview

The Aquera platform allows your organization to scale complex data management by automating the synchronization of data across resource types. This guide covers the end-to-end tasks in setting up a data synchronization to sync the Employees resource of UKG Ready to the Employees resource of QuickBooks Desktop Enterprise.

Prerequisites

The following are the prerequisites:

  • Admin Access to QuickBooks Desktop Enterprise: You must have admin access to your QuickBooks Desktop Enterprise instance. You can visit the QuickBooks Desktop Enterprise portal to get access to an QuickBooks Desktop Enterprise instance.
  • Tenant Account in the Aquera Admin Portal: You must have a tenant account with admin access in the Aquera Admin portal. Visit https://admin.aquera.io and register to get a tenant account. Contact your Aquera account manager to get the Aquera tenant access.
    Note: If your organization is in the EU region, use the following URL to register: https://admin.eu.aquera.io/
  • Configure Application Connectors in the Aquera Admin Portal: You need to configure following application connectors in the Aquera Admin portal to integrate with QuickBooks Desktop Enterprise:
    • UKG Ready: Refer to the UKG Ready Configuration Guide for details.
    • QuickBooks Desktop Enterprise: Refer to the QuickBooks Desktop Enterprise Configuration Guide for details.

Data Synchronization Setup

Aquera's data synchronization automatically imports employees from UKG Ready and provisions them in QuickBooks Desktop Enterprise.  Aquera provides pre-configured synchronizations created for specific use cases as templates. You can use these templates to create a data synchronization.

Create Data Synchronization

Perform the following steps to create a new data synchronization:

  1. Log in to the Aquera Admin portal.
  2. Navigate to Synchronizations from the left side panel.
  3. Click Create Synchronization.
  4. Select UKG Ready Employees to QuickBooks Desktop Enterprise Employees.
  5. Enter a name and a description for the synchronization.
  6. Select your UKG Ready and QuickBooks Desktop Enterprise connector instances.
  7. Click Continue. A new page is displayed to configure the synchronization.

Run Script Synchronization

You can click the Execute button at the top right of the data synchronization to run the data synchronization manually. You can also schedule the data synchronization to run automatically at specified intervals by configuring a frequency in the Schedule tab. 

You can navigate to Logs  > Synchronizations in the Aquera Admin portal to view the logs of the synchronization execution.
